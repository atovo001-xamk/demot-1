Demo 8: Käyttäjähallinta

Tämä on demo REST API -palvelinsovelluksen React-asiakassovelluksen yhteistoiminnasta. Lisäyksenä edelliseen demoon on käyttäjän autentikonti ja käyttäjätietojen hallinta. Käynnistä Reactin kehityspalvelin suorittamalla komento 'npm start' client-kansiossa

Testaa asiakassovelluksen toiminnallisuutta selaimella osoitteessa http://localhost:3000. Palvelinsovellusta voit testata Postman-ohjelmalla osoitteessa http://localhost:3008/api/ostokset, jos laitat tokenin osaksi pyyntöjä headers-välilehdellä (Authorization) samalla tavoin kuin React-sovelluksessa
